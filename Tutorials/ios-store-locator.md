<p>This guide will walk you through how to use our <strong>iOS Store Locator starter kit</strong> to create a custom store locator map that can be integrated into any iOS application. You'll be able to browse several locations, select a specific location to view more information, and retrieve a route from a user's location to any of the store locations. You can start with one of five different themes and customize everything from store location data to marker icons and individual store cards.</p>
<p>{{<a target="_blank" rel="noopener noreferrer" href="/mapbox/help/blob/publisher-production/help/img/ios/store-locator-final-product.gif"><img alt="animated GIF of a store locator application on an iOS device" src="/mapbox/help/raw/publisher-production/help/img/ios/store-locator-final-product.gif" style="max-width:100%;"></a>}}</p>
<h2><a id="user-content-getting-started" class="anchor" aria-hidden="true" href="#getting-started"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Getting started</h2>
<p>The iOS Store Locator starter kit runs on iOS 9.0 and above. It can be used with code written in Swift 3.1 and above using Xcode 9. Here are the resources youâll need before getting started:</p>
<ul>
<li><strong>Starter kit files</strong>. Download or clone the <a href="https://github.com/mapbox/store-locator-ios/">iOS Store Locator starter kit</a> from GitHub. This includes all the necessary files for a functional Xcode project.</li>
<li><strong>A Mapbox account and access token</strong>. Sign up for an account at <a href="https://www.mapbox.com/signup/" rel="nofollow">mapbox.com/signup</a>. You can find your <a href="/mapbox/help/blob/publisher-production/help/how-mapbox-works/access-tokens">access tokens</a> on your <a href="https://www.mapbox.com/account/" rel="nofollow">Account page</a>. You will add your access token to your Info.plist file.</li>
<li><strong>A physical or <a href="https://help.apple.com/simulator/mac/current/#/deve44b57b2a" rel="nofollow">simulated iOS device</a></strong>.</li>
<li><strong>Optional: SVG editing tool</strong>. If you'd like to create custom icons, you can use a program like <a href="http://sketchapp.com" rel="nofollow">Sketch</a>, <a href="http://figma.com" rel="nofollow">Figma</a>, or Adobe Illustrator.</li>
</ul>
<h2><a id="user-content-set-up-the-starter-kit" class="anchor" aria-hidden="true" href="#set-up-the-starter-kit"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Set up the starter kit</h2>
<p>The <a href="https://github.com/mapbox/store-locator-ios/">iOS Store Locator starter kit</a> includes all the source files needed to build a store locator. The kit includes:</p>
<ul>
<li>Five UI theme variations.</li>
<li>Sample GeoJSON data with store locations.</li>
<li>Code for retrieving directions with the <a href="https://docs.mapbox.com/api/navigation/#directions" rel="nofollow">Mapbox Directions API</a> and displaying a navigation route line on the map.</li>
</ul>
<h3><a id="user-content-install-dependencies" class="anchor" aria-hidden="true" href="#install-dependencies"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Install dependencies</h3>
<p>Start by installing the project's dependencies using CocoaPods. If you're new to CocoaPods, read the <a href="https://guides.cocoapods.org/using/getting-started.html" rel="nofollow">CocoaPods documentation</a> before getting started. You'll need to run <code>pod update</code> in the directory containing the starter kit files to add the necessary pods (<strong>Mapbox-iOS-SDK</strong> and <strong>MapboxDirections.swift</strong>) to your project.</p>
<p>Once youâve run <code>pod update</code>, work out of the space <code>mapbox-store-locator.xcworkspace</code> (not <code>mapbox-store-locator.xcodeproj</code>). The <code>mapbox-store-locator.xcworkspace</code> includes the necessary dependencies. The easiest way to do this from the command line is to use <code>open mapbox-store-locator.xcworkspace/</code> after your pods have been installed.</p>

<p>In this tutorial you'll work with the fifth theme displayed in the theme picker preview, the <code>neutralTheme</code>. This theme features the <a href="https://www.mapbox.com/maps/streets/" rel="nofollow">Mapbox Streets</a> style and a house icon at each store location. To set this theme:</p>

<h2><a id="user-content-add-your-data" class="anchor" aria-hidden="true" href="#add-your-data"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Add your data</h2>
<p>The starter kit contains a GeoJSON file called <code>stores.geojson</code>  where youâll find all the current store locations that are visible on the map. <a href="/mapbox/help/blob/publisher-production/help/glossary/geojson">GeoJSON</a> is a file format for geospatial data and a subset of the JSON format. In this section, you'll update the sample data to reflect your actual store locations.</p>
<p>Locate the <code>stores.geojson</code> file and take a look at how the data is formatted. Notice that each store location is a separate GeoJSON <em>feature</em> in the file. Each feature has four <code>properties</code> describing a few characteristics about the store location and <code>geometry</code> specifying that the feature is a single point and where that point is located in the world.</p>
<p>If you have store location data available in GeoJSON form, you can delete the current data and replace it with your own.</p>
